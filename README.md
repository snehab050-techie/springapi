## Overview
This is a Spring Boot-based REST API application for managing student data. The project provides endpoints for CRUD operations on student entities, built with Java and Spring Framework.

## Features
- RESTful API endpoints for student management
- Data transfer objects (DTOs) for request/response handling
- Entity mapping with JPA/Hibernate
- Service layer for business logic
- Controller layer for HTTP request handling

## Prerequisites
- Java 17 or higher
- Maven 3.6+
- Spring Boot 3.x
